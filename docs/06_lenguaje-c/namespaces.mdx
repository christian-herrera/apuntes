# NameSpace


## ¿Qué es un namespace?
Un namespace es un contenedor de nombres que agrupa:

- Funciones
- Variables
- Clases
- Tipos
- Constantes

Su objetivo principal es: ***Evitar colisiones de nombres y organizar el código***, no crea objetos, no ocupa memoria por sí mismo, y no es OOP.

Un ejemplo sencillo es el siguiente:

<Tabs>
    {/* MARK: Pestaña main.cpp */}
    <TabItem value="main.cpp" label="main.cpp" default>

        ```cpp
        int main() {
        ```

    </TabItem>

    {/* MARK: Pestaña sensor.cpp */}
    <TabItem value="sensor.cpp" label="sensor.cpp" default>
        ```cpp
        #include "sensor.h"

        // Namespace anónimo para variables privadas
        namespace {
            bool initialized = false;
        }

        // Definición de las funciónes públicas
        namespace Sensor {
        void Sensor::begin() {
            // Inicialización del sensor
            initialized = true;
        }
        }
        ```
    </TabItem>
