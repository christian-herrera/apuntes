Hashing de Claves

En PHP, el hashing de claves es una práctica esencial para garantizar la seguridad de las contraseñas almacenadas en una base de datos. PHP proporciona funciones integradas que facilitan este proceso de manera segura y eficiente.

Funciones Principales​

  • password_hash(): Genera un hash seguro de una clave.
  • password_verify(): Verifica si una contraseña ingresada coincide con un hash generado por password_hash().
  • password_needs_rehash(): Verifica si un hash necesita ser actualizado a un nuevo algoritmo o costo.
  • password_get_info(): Proporciona información sobre el algoritmo y las opciones utilizadas para crear un hash.

Y algunas de las constantes mas importantes son:

  • PASSWORD_DEFAULT: Utiliza el algoritmo de hashing predeterminado (actualmente BCRYPT).
  • PASSWORD_BCRYPT: Utiliza el algoritmo BCRYPT para el hashing.
  • PASSWORD_ARGON2I: Utiliza el algoritmo Argon2i para el hashing.
  • PASSWORD_ARGON2ID: Utiliza el algoritmo Argon2id para el hashing (mezcla lo mejor de Argon2i + Argon2d).

Ejemplo de Uso​

$clave_raw = "perro1234";

// Generar un hash de la clave
$hash = password_hash($clave, PASSWORD_DEFAULT);

// Verificar la clave ingresada contra el hash almacenado
echo password_verify($clave_raw, $hash) ? "¡Clave verificada!" : "Clave incorrecta.";
