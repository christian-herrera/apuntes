Estructuras

Estructura del proyecto​

Al igual que en otros lenguajes de programación, en Python es importante mantener una estructura organizada para los proyectos. A continuación, se muestra una estructura básica recomendada para un proyecto en Python:

mi_proyecto/
├ .venv/                  # Entorno virtual
├ main.py                 # Archivo de entrada principal
│   
├  core/
│   ├ __init__.py         # Hace que src sea un paquete
│   ├ module1.py          # Módulo con funciones
·   ├ module2.py          # Módulo con funciones
·   ·
·   ·
·

De esta forma, cuando se requiere importar funciones que están dentro de los módulos, se utiliza:

  • main.py
  • module1.py
  • module2.py
from core.module1 import saludar
from core.module2 import despedir

def main():
    saludar()
    despedir()

if __name__ == "__main__":
    main()
def saludar():
    print("Hola desde el módulo 1")
def despedir():
    print("Adiós desde el módulo 2")

Sintaxis​

Estructuras de Control​

En Python, la sintaxis es sencilla y fácil de aprender. A continuación, se presentan algunas de las estructuras básicas del lenguaje:

  • if-elif-else
  • While
  • Match
  • For

Recordar que no se utilizan llaves {} para definir bloques de código, sino que se usa la indentación (espacios o tabulaciones).

x = 10

if x > 0:
    print("positivo")
elif x == 0:
    print("cero")
else:
    print("negativo")

La versión inline tiene la estructura:

print("positivo") if x > 0 else print("cero") if x == 0 else print("negativo")
count = 0
while True:
    count += 1
    if count == 5:
        break
    if count >= 3:
        continue
    print(count)   # Imprime 1 y 2 (Salta 3 y 4)

match es similar a switch en otros lenguajes. El valor por defecto se maneja con _.

valor = 2

match valor:
    case 1:
        print("Uno")
    case 2:
        print("Dos")
    case 3:
        print("Tres")
    case _:
        print("Otro valor")

En Python, el bucle for se utiliza para iterar sobre elementos de una secuencia (como listas, tuplas, diccionarios, conjuntos o cadenas). Recorrer un valor incremental similar a otros lenguajes se hace con la función range().

for i in range(5):
    print(i)

Otro ejemplo, es recorrer los elementos de una lista:

frutas = ["manzana", "banana", "cereza"]

for fruta in frutas:
    print(fruta)

Variables y Tipos de Datos​

En Python, no es necesario declarar el tipo de una variable explícitamente. El intérprete asigna el tipo automáticamente según el valor asignado. Algunos tipos de datos comunes en Python son:

  • None
  • Bool
  • Int/Float
  • Str
  • List
  • Tuplas
  • Diccionario
  • Conjuntos

Representa la ausencia de valor o un valor nulo.

var = None
print(var)  # Salida: None

Representa valores booleanos: True o False.

es_verdadero = True
es_falso = False
print(es_verdadero)  # Salida: True
print(es_falso)      # Salida: False

Representan números enteros y números de punto flotante (decimales).

entero = 10
flotante = 3.14
print(entero)    # Salida: 10
print(flotante)  # Salida: 3.14

Representa cadenas de texto.

texto = "Hola, Mundo!"
print(texto)  # Salida: Hola, Mundo!

Algunas operaciones útiles con cadenas:

# Para usar como condicionales:
s.startswith("a")             # Devuelve True si la cadena empieza con "a"
s.endswith("z")               # Devuelve True si la cadena termina con "z"

# Para manipular cadenas:
s.lower()                     # Convierte a minúsculas
s.upper()                     # Convierte a mayúsculas
s.split(",")                  # Divide la cadena en una lista usando "," como separador
s.replace("viejo", "nuevo")   # Reemplaza "viejo" por "nuevo"

Representa una colección ordenada y mutable de elementos.

frutas = ["manzana", "banana", "cereza"]
lst = [1, 2, 3]

Estos valores se pueden recorrer con:

for fruta in frutas:
    print(fruta)

Representa una colección ordenada e inmutable de elementos.

coordenadas = (10.0, 20.0)
tupla = (1, 2, 3)

A diferencia de las listas, las tuplas no pueden modificarse después de su creación.

Representa una colección desordenada de pares clave-valor.

persona = {
    "nombre": "Juan",
    "edad": 30,
    "ciudad": "Madrid"
}

Para acceder a los valores, se utiliza la clave correspondiente:

print(persona["nombre"])  # Salida: Juan

Representa una colección desordenada de elementos únicos.

numeros = {1, 2, 3, 4, 5}

Los conjuntos son útiles para realizar operaciones matemáticas como unión, intersección y diferencia.

a = {1, 2, 3}
b = {3, 4, 5}

union = a | b               # {1, 2, 3, 4, 5}
interseccion = a & b        # {3}
diferencia = a - b          # {1, 2}

Valores Falsy y Truthy​

En Python, ciertos valores se consideran "falsy" (evaluados como False en un contexto booleano), mientras que otros se consideran "truthy" (evaluados como True).

Todos los falores son truthy, excepto los siguientes valores falsy:

False
None
0
0.0
0j
""
b""
bytearray()
[]
()
{}
set()
range(0)

¿Cuando nó utilizar valores falsy o truthy? Cuando importa distinguir:

  • None vs 0
  • "" vs "0"
  • Colección vacía vs no inicializada
