# Funciones
Una función en Python es un bloque de código reutilizable que realiza una tarea específica. Las funciones ayudan a organizar el código, mejorar su legibilidad y facilitar el mantenimiento.

```python
def sumar(a, b):
    return a + b

resultado = sumar(3, 5)
print("La suma es:", resultado)
```

## Retorno de Valores
Se pueden retornar múltiples valores desde una función utilizando tuplas:

```python
def dividir(a, b):
    return a // b, a % b

q, r = dividir(10, 3)
print("Cociente:", q)
print("Resto:", r)
```

### Argumentos
Las funciones pueden aceptar diferentes tipos de argumentos:
```python
def mostrar_info(nombre, edad=30, *args, **kwargs):
    print("Nombre:", nombre)
    print("Edad:", edad)
    print("Args adicionales:", args)
    print("Kwargs adicionales:", kwargs)

mostrar_info()                                                          #1
mostrar_info("Luis")                                                    #2
mostrar_info("Maria", 28)                                               #3
mostrar_info("Carlos", 35, "Estudiante", "Deporte")                     #4
mostrar_info("Ana", 25, "Estudiante", ciudad="Madrid", pais="España")   #5
```

Es importante entender como funcionan las diferentes llamadas a la función `mostrar_info`:
