Inicialización

Para hacer uso de React se requiere instalar Node JS. Esto se logra ejecutando el script que devuelve la web en funcion del sistema operativo.

La forma más práctica de crear un proyectn nuevo es utilizando Vite el cual permite ejecutar:

npm create vite@latest my-project
